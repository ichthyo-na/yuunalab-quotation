# YUUNALAB+ Quotation System

A lightweight, single-file quotation/estimate generator designed for photography, video production, and creative studios.

## Features

- **Single HTML file** - No build tools, no dependencies, just open and use
- **Bilingual (EN/CN)** - Full English & Chinese support for international productions
- **Multiple project types** - Photography, Video, Full Production, AI/3D
- **Smart templates** - Pre-built role & fee templates (photographers, stylists, editors, 3D artists, etc.)
- **PDF export** - One-click export to professional PDF quotations
- **Equipment & Props lists** - Dedicated tabs for tracking gear and props
- **Auto-calculation** - Real-time subtotals, tax, and grand total
- **Customizable branding** - Logo upload, accent color, brand name
- **Local storage** - Auto-saves your work in the browser
- **Save/Load JSON** - Export and import quotation data

## Quick Start

1. Download `index.html`
2. Open it in your browser
3. Start creating quotations

That's it. No server, no installation, no configuration.
